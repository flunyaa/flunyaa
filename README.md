# Flunyaa

## 介绍

基于 Flutter 开发的 [Nyaa](https://nyaa.si) 客户端，可以进行浏览、搜索、下载种子文件、通过磁链打开 Torrent 应用。

## 国际化请求 🔍

本项目将在基础功能稳定后启动界面国际化进程。
